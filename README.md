# USDA Beef Price Dashboard

## Project Overview
This project pulls live beef price data from the USDA API and builds 
an interactive dashboard with price trends and ARIMA forecasting.

## Features
- Live data from USDA Agricultural Marketing Service API
- 180 days of historical beef price data
- Week-over-week % price change analysis
- ARIMA forecasting model with 95% confidence intervals
- Model accuracy up to 99.45%

## Technologies Used
- Python
- Pandas
- Plotly
- Statsmodels / pmdarima
- USDA MPR Datamart API

## How to Run
1. Open the notebook in Google Colab
2. Add your USDA API key
3. Run all cells
